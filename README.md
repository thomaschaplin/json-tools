# JSON Tools

[JSON Tools Website](http://json.thomaschaplin.me)

A simple website to manage JSON files

Accepts [JSON](https://www.json.org/) or [JSON5](https://json5.org/) format

* Drag 'N' Drop Files
* Pretty Print
* Minify
* Validate JSON
* Validate JSON5
* Download
* Copy
* Clear
